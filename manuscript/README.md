# Manuscript

The thesis manuscript and figures.

## Organisation

- `chapters/` — chapter sources (LaTeX or Markdown).
- `figures/` — figures referenced from chapters.

## Source format

The manuscript is currently maintained in DOCX (with tracked changes for revisions). The transition to a single source format (LaTeX or Markdown) is a Phase 6 deliverable.

## Cross-references with the repository

Every chapter cross-references identifiers from the living documents under `docs/`. The check that all identifiers used in the manuscript are defined in `docs/` is part of the pre-Gate procedure (to be added to `tools/check_traceability.py` in Phase 1).

## Phase status

- Phase 0: outline established; abstract, RQs, objectives, contributions and conclusions drafted.
- Phase 1: Chapter 5 sections (HARA, SRS, Cage Specification) written.
- Phase 6: full manuscript closed and submitted.
